# CareerPilot AI 🚀

CareerPilot AI is an AI-powered interview preparation and resume-tailoring assistant. It analyzes a candidate's resume, self-description, and target job description to generate a comprehensive interview preparation report (complete with tailored technical and behavioral questions, day-by-day roadmaps, skill gap analysis, and match scores) and generates a downloadable, job-tailored resume PDF.

---

## 📂 Project Structure

The project is structured as a monorepo with separate frontend and backend directories:

```text
careerpilot-ai/
├── backend/          # Express API server powered by Node.js & MongoDB
│   ├── src/
│   │   ├── config/      # Database & configuration files
│   │   ├── controller/  # Express route handlers
│   │   ├── middleware/  # Authentication & file upload middleware
│   │   ├── models/      # Mongoose schemas (User, InterviewReport)
│   │   ├── routes/      # Express API routers
│   │   └── services/    # AI GenAI Integration (Gemini) & PDF Services
│   └── server.js        # Main server entrypoint
│
└── frontend/         # Vite + React client app
    ├── src/
    │   ├── feature/
    │   │   ├── auth/       # Authentication context, hooks & pages
    │   │   └── interview/  # Interview report dashboards, hooks & forms
    │   └── main.jsx        # App mounting point
    └── index.html
```

## ✨ Features

- **AI Interview Prep Generator**: Custom lists of technical and behavioral questions tailored to your target job, including detailed explanations of the interviewer's intent and high-scoring model answers.
- **Match Score & Skill Gap Analysis**: Detailed percentage match score and key skill gap breakdown (labeled by severity: low, medium, high) comparing your profile to the target job.
- **Preparation Roadmap**: A day-by-day learning schedule and customized preparation plan to get you ready for the interview step-by-step.
- **AI Resume PDF Tailoring**: Generates a professional, job-tailored, ATS-friendly resume as a downloadable PDF, generated by Gemini and converted cleanly using Puppeteer.

---

## 🛠️ Tech Stack

### Backend
- **Node.js & Express**: High-performance backend API framework.
- **MongoDB & Mongoose**: Flexible document database for storing user accounts and generated interview reports.
- **Google GenAI SDK (`@google/genai`)**: Uses state-of-the-art **Gemini 2.5** models for interview plan generation and resume building.
- **Puppeteer**: Automated headless browser for pixel-perfect PDF rendering from AI-generated HTML resumes.
- **PDF Parse & Multer**: Parsing and extracting text from uploaded resume PDF files.

### Frontend
- **React.js (Vite)**: Quick and reactive single-page app interface.
- **Tailwind CSS**: Modern utility-first CSS styling.
- **Axios**: HTTP client for API communication.

## 🚀 Getting Started

### Prerequisites
Make sure you have the following installed:
- [Node.js](https://nodejs.org/) (v18+)
- [MongoDB](https://www.mongodb.com/) (running locally or a MongoDB Atlas URI)

---

### Backend Setup

1. Navigate to the `backend` directory:
   ```bash
   cd backend
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Create a `.env` file in the `backend` folder and populate it with the following:
   ```env
   PORT=8080
   MONGO_URI=mongodb://localhost:27017/careerpilot
   JWT_SECRET=your_super_secret_jwt_key
   GOOGLE_GENAI_API_KEY=your_gemini_api_key
   ```
   *(Ensure you replace the values above with your actual MongoDB URI and Gemini API Key).*

4. Start the backend development server:
   ```bash
   npm run dev
   ```

---

### Frontend Setup

1. Navigate to the `frontend` directory:
   ```bash
   cd ../frontend
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the frontend development server:
   ```bash
   npm run dev
   ```

---
